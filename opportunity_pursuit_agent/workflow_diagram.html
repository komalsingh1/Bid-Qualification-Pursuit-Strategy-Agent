Phase breakdown
Phase 1
Intake
  • Parse PDF/DOCX/portal
  • Extract requirements
  • Tag mandatory vs. scored
  • Identify deadlines & metadata
Phase 2
Analysis
  • Match reqs to capabilities
  • Flag gaps & missing certs
  • Surface competitive context
Phase 3
Memory Retrieval
  • Retrieve similar past bids
  • Search proposal library
  • Identify SME owners
Phase 4
Decision Engine
  • Score 0–100 bid confidence
  • Auto-proceed ≥ 70
  • Escalate to human 40–69
  • No-bid below 40
Phase 5
Output Package
  • Go / No-Go report
  • Gap action plan + owners
  • Top 3 win themes
  • Submission checklist
